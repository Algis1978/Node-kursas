Node.js

Įvadas

Node.js yra atvirojo kodo, daugiaplatformė JavaScript veikimo aplinka, orientuota į tinklo ir serverių programas.

Node.js veikia Windows OS, Mac OSX, Linux, Unix ir kitose operacinėse sistemose.

Idėja sukurti Node.js 2009 m. kilo programuotojui Ryan Dahl. Node.js viešai pristatyta 2009 m. gruodį.

Vadovo susitarimai

Prieš padedant naudotis Node.js jūs turite būti susipažinę su JavaScript pogramavimo pagrindais ir turėti neblogus bendrus naudojimosi kompiuteriu įgūdžius.

Toliau vadove bendrai gerai žinomi pavadinimai (pvz., Windows) nebus rašomi kabutėse, reikiamas surinkti kodas bus rašomas pasvyruoju šriftu.

Toliau naudojamas JavaScript trumpinys „JS“ ir Node.js trumpinys „Node“.

Node.js paruošimas darbui

Atsisiųsti Node.js paketą ir jį instaliuoti galima iš oficialaus projekto tinklalapio: https://nodejs.org/en/download/releases/

Mes galime pasitikrinti Node.js versiją Windows sistemos 'Command Prompt' lange surinkę node -v ir nuspaudę 'Enter'.

Darbui su Node.js reikalingas teksto redaktorius. Jų pavyzdžiai: Atom, NotePad++, VS Code, Sublime ir kt.

Instaliavus teksto redaktorių Node.js komandų veikimą galima stebėti 'Terminal' lange atskirai nuo kodo redagavimo lango.

'Terminal' lange surinkus node -v komandą ir nuspaudusFpir 'Enter' jums ten pat bus parodyta esamos Node.js versijos numeris.

Pirmos programos parašymas

Savo bylų sistemoje susikurkite naują katalogą pasirinktu pavadinimu.

Jame susikurkite JS bylą, pvz., pavadinimu pirmas.js.

Šią bylą atsidarykite su teksto redaktoriumi (galite katalogą ir JS bylą susikurti iš teksto redaktoriaus interfeiso.

Atsidarytos bylos kodo lange surinkite:

console.log("Sveiki, mano vardas ...");

Išsaugokite bylą (Ctrl+S klavišų kombinacija Windows sistemoje). Po to terminalo lange surinkite node pirmas.js ir nuspauskite 'Enter'. Terminalo lange bus pavaizduota užrašas „Sveiki, mano vardas ...“ su jūsų vietoj trijų taškų įvestu vardu.
