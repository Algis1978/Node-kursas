Klaidų ieškojimas su try...catch

Klaidų radimui JS turi įdiegtą try...catch (iš anglų. k. „tikrinti“ ir „pagauti“) steitmentą, kuris ieško kode klaidų.

Dirbtinės klaidos sukūrimas

Klaidos metimui JS įdiegtas throw (iš anglų. k. „mesti“) steitmentas. Šiuo steitmentu sukuriamas dirbtinis klaidos pranešimas.

JS įdiegtas globalus Error objektas, kuriuo galima sukurti klaidos objektą:

const klaida = new Error ("Tai yra klaida");
console.log( typeof(klaida) )

Surinkite aukštesnį kodą ir po to meskite dirbtinę programos klaidą:

throw klaida
console.log("Tai programos tęsinys");

Vykdant programą klaidos pranešimas bus išvestas terminale. Jūs pastebėsit, kad tolimesnis programos kodas nebevykdomas ir užrašo "Tai programos tęsinys" nebepamatysite.

Taip yra todėl, kad programai nėra nurodyta, ką daryti su joje pasitaikiusia klaida. Tam pagelbės try...catch steitmentas.

Steitmento try...catch panaudojimas

Meskime aukščiau sukurtą klaidą į try...catch steitmentą:

try {throw klaida;}
catch (klaidosObjektas) {console.log(klaidosObjektas);}
console.log("Tai programos tęsinys");

Įvykdžius šią programą jūs pamatysite klaidos pranešimą terminale ir taip pat pamatysite, kad tolimesnis programos kodas yra vykdomas.

Steitmento try...catch panaudojimo pavyzdys

Sukurkime neegzistuojančią tekstinę bylą ir parašykime programą, kuri ją perskaitytų. Ji negalės to padaryti ir mes klaidą.

const klaidosByla = "klaidosByla.txt";
const klaidosBylosTekstas = myFs.readFileSync(klaidosByla, "utf8");
console.log(klaidosBylosTekstas);

Įvykdžius šią programą terminale bus išvestas klaidos pranešimas ''no such file or directory'' („nėra tokios bylos arba katalogo“).

Praktinio pritaikymo vardan parašykime funkciją, kuri perskaitytų bylos tekstą ir išvestų jos turinį į terminalą. Klaidos skaityme atveju išvestų klaidos pranešimą ir grąžintų null objektą:

function bylosSkaitytuvas (byla) {
try{ const bylosTekstas = myFs.readFileSync(byla, "utf8");
console.log("Bylos tekstas:");
return bylosTekstas;}
catch (err) {
console.log("Klaida bylos skaityme");
console.log(err);}
return null;}

Dabar šią funkciją galima naudoti su bylomis. Savarankiškai sukurkite savo bylas ir leiskite su programa bylosSkaitytuvas() jas perskaityti.

Jūs pastebėsit, kad klaidos objektą funkcijos kode nurodo žodelis err. Yra įprasta JS kode taip žymėti klaidos objektą (iš anglų k. error reiškia „klaida“).

Apibendrinimas

Steitmentas try...catch leidžia kontroliuoti klaidas kode ir nustatyti, ką joms pasitaikius programai daryti.
