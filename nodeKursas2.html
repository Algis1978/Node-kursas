Node moduliai

Node moduliai (module) yra kitos JS bylos, kurias galima panaudoti kuriant savo programas. Jie gali būti suinstaliuoti kartu su Node paketu, susikurti savo arba sukurti kitų programuotojų. Pastaruoju atveju šios bylos vadinamos bibliotekomis (library) arba paketais (package).

NPM inicijavimas

Instaliuoti Node modulius galima su Node paketų menedžeriu NPM (Node Package Manager)

Inicijuokite NPM veikimą su komanda terminalo eilutėje:

npm init

Nuspauskite 'Enter' ir jums terminale bus pasiūlyta įvesti jūsų projekto savybių vertes. Jei nenorite į tai gilintis, ties kiekvienu punktu nuspauskit 'Enter'. Trumpesnis kelias inicijuoti NPM yra surinkus kodą npm init -y

Po šios komandos jūsų darbo kataloge atsiras byla package.json, kurioje bus surašytos jūsų projekto savybės.

Node modulių instaliavimas ir susiejimas

Vienas iš Node modulių yra pavadinimu „path“. Suinstaliuokite jį su komanda:

npm install path

Jūs pastebėsit, kad byloje package.json atsirado nauja savybė „dependencies“ su verte „path“. Tai rodo, kad jūsų projektui vykdyti reikės Node modulio „path“ (jei jis bus realiai panaudotas).

Taip pat jūsų darbo kataloge atsiras package-lock.json byla, kuri seks modulių versijų suderinamumą.

Ir jūsų darbo kataloge atsiras „node_modules“ katalogas su Node moduliais.

Moduliai su JS byla susiejami su require raktažodžiu. Surinkite šį kodą bylos pirmas.js kodo eilutėje ir išsaugokite bylą:

const path = require("path");

Šiuo kodu kintamajam „path“ buvo priskirtos modulio „path“ savybės, kurias galima naudoti tolimesniame kode

Modulio savybių panaudojimas

Modulis „path“ skirtas darbui su katalogų ir bylų pavadinimais. Pavyzdžiui, jo savybė .extname grąžina bylos tipo praplėtimo užrašymą. Toliau byloje pirmas.js surinkite šį kodą:

const byla = "pirmas.html";
console.log(path.extname(byla));

Patikrinkite rezultatą terminalo lange su komanda node pirmas.js. Jums turėtų būti parodytas rezultatas: .html

Kaip jūs pastebėjote, dirbant su Node nuolat reikia šokinėti tarp JS bylos kodo ir teminalo langų reaktoriuje, tad turėkite juos šalia vienas kito.

Taipogi, kad kiekvieną kartą nerinkti tos pačios komandos terminalo lange redaktoriai turi trumpojo iššaukimo komandą, kuri priklauo nuo redaktoriaus nustatymų, pvz., tai gali būti rodyklėlės viršun klavišo nuspaudimas, kuris jums išves paskutinę komandą, šiuo atveju node pirmas.js.

Trečiųjų šalių moduliai

Kitų programuotojų sukurtas bibliotekas galima susieti su savo JS projektais lygiai taip pat, kaip Node modulius.

Sakykime, aš noriu panaudoti savo projekte modulį pavadinimu „faker“, kurs generuoja netikrus žmonių duomenis. Instaliuoju terminale su NPM:

npm install path

Jūs turėtumėte rasti „faker“ katalogą tarp Node modulių ir package.json byloje tarp „dependencies“ savybių atsiras „faker“.

NPM instaliuoja paskutines Node modulių versijas, tačiau naudotojas gali specifiškai nurodyti jo norimą versiją už @ ženklo, pvz., faker@5.1.3
