Komandinės eilutės programos

Node leidžia vartotojui sąveikauti su JS programomis įvedant duomenis komandinėje eilutėje.

Šis procesas apima dvi dalis: JS programą ir vartotojo komandinėje eilutėje įvedamų duomenų (argumentų JS programai) apdorojimo programą.

JS programos pavyzdys

Sukurkime programą, kuri išves atsitiktinius sveikuosius skaičius nuo 1 iki 100. Tai galima padaryti panaudojus „Math.random()“ ir „Math.ceil()“ metodus:

function atsitiktinisSkaicius ( max = 100 )
{ return Math.ceil( Math.random()*max ); };
console.log(atsitiktinisSkaicius())

Pasitikrinkite terminale, ar išties programa grąžina sveikuosius skaičius tarp 1 ir 100.

Komandinės eilutės duomenų programos pavyzdys

Vartotojo įvestus duomenis galima paimti iš komandinės eilutės naudojant process.argv metodą. Objektas process yra globalus Node objektas. Su .argv metodu yra pateikiamas masyvas, turintis du elementus: kelią iki Node įdiegimo katalogo ir kelią iki darbinės JS bylos. Įsitikinkite tuo terminale:

console.log(process.argv);

Kai vartotojas komandinėje eilutėje pateikia papildomus duomenis, juos tampa įmanoma atskirti ir panaudoti JS programoje. Vietoj node pirmas.js komandinėje eilutėje surinkite node pirmas.js TAIP NE ir patikrinkite, ką terminale išves process.argv metodas.

Jūs pamatysite, kad išvestame masyve atsirado trečias ir ketvirtas elementai (nr. 2 ir nr. 3), kurių vertės yra "TAIP" ir "NE" atitinkamai.

Sakykime, norime, kad vartotojas galėtų kontroliuoti atsitiktinio skaičiaus generatoriaus intervalo viršutinę ribą „max“ iš komandinės eilutės. Tam prireiks trečiojo elemento process.argv masyve, ir jį galima išvesti destruktūrizacijos pagalba:

const [ pirmas, antras, vartotojoSkaicius ] = process.argv;
console.log(vartotojoSkaicius);

Dabar vartotojo įvestą skaičių nukreipus į atsitiktinio skaičiaus funkcijos parametrą „max“, šios funkcijos vertės bus tarp 1 ir vartotojo įvesto skaičiaus. Surinkite sekančią kodo eilutę, kad tai matytumėte:

console.log( atsitiktinisSkaicius(vartotojoSkaicius) );

Sakykime, tai skaičius 20. Įveskite komandą terminalo eilutėje:

node pirmas.js 20

Kartodami šią komandą jūs turite pastebėti, kad atsitiktinio skaičiaus funkcijos verčių ribos svyruoja tarp skaičių 1 ir 20.
