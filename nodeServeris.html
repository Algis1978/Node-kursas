Serverio sukūrimas

Atskirame kataloge susikurkime naują projektą (inicijuokime su npm install ir npm init komandomis. Byloje package.json įveskime "type" vertę "module". Po to susikurkime JS bylą, pvz., serveris.js ir su ja ssusiekime Net modulį:

import * as net from "net";

Po to sukurkime serverį su komanda:

URL adreso perskaitymas

protokolas://host:port/path?query-string#fragment-id

Kur prisijungimo protokolas (http, https, ftp, ssh)

host - svetainės serveris (www.adresas.lt)

port - prisijungimo taškas (3000, 80, 443 ir kt.)

path - specifinis resursas, kurį nori pasiekti užklausėjas,

query-string - papildomi duomenys serverio pusės programoms(skriptams).

fragment-id - nurodo specifinę vietą puslapyje, kurią reikia pasiekti.

Nurodžius IP:porto nr. (192.162.3.12:8080), serveris pasiekiamas iš išorės.

socket.write(k, "utf8") nusiųs atsakymą į užklausą. \r\n kodas tekste nurodo naują eilutę.
